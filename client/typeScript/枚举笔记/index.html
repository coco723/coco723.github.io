# 枚举笔记

# 数字枚举

数字枚举可以被混入到计算过的和常量成员。简短地说，不带初始化器的枚举或者被放在第一的位置，或者被放在使用了数字敞亮或其他长量初始化了的枚举后迷啊。换句话说，下面的情况是被允许的：

enum E {
    A = getSomeValue(),
    B, // error! 'A' is not constant-initialized, so 'B' needs an initializer
}
